A procedural flower generation tool built in Houdini that allows artists to create a wide variety of stylized flowers through intuitive parameters.

## Overview

This tool was designed to give artists control over flower generation while maintaining the efficiency of procedural workflows. Users can adjust petal count, shape, color gradients, stem curvature, and more.

## Features

- Adjustable petal count and arrangement
- Customizable petal shapes and curves
- Procedural color variation and gradients
- Stem and leaf generation
- Export-ready geometry for game engines

## Technical Approach

The tool uses Houdini's node-based system to create a flexible, non-destructive workflow. VEX expressions drive the procedural variations, allowing for both randomization and precise art direction.
