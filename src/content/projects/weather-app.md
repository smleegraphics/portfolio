A weather dashboard application that displays current conditions and forecasts for any location. Built with React and integrates with the OpenWeatherMap API.

## Features

- Search for any city worldwide
- Current conditions with animated icons
- 7-day forecast with temperature graphs
- Hourly breakdown for the next 24 hours
- Save favorite locations

## Technologies

- React 18 with hooks
- Chart.js for visualizations
- Tailwind CSS for styling
- OpenWeatherMap API
