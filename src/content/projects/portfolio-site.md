This portfolio website was built to showcase my work in a clean, performant, and accessible way. Built with modern web technologies, it prioritizes fast load times and a great user experience.

## Tech Stack

- **Astro** - Static site generator with excellent performance
- **Tailwind CSS** - Utility-first styling
- **TypeScript** - Type safety throughout
- **Content Collections** - Type-safe Markdown content management

## Features

- Responsive design that works on all devices
- Dark mode support (coming soon)
- Image lightbox with keyboard navigation
- RSS feed for blog posts
- SEO optimized with Open Graph tags

## Performance

The site achieves a perfect 100 score on Lighthouse for Performance, Accessibility, Best Practices, and SEO. All images are optimized and lazy-loaded.
