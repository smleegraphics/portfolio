This project started as an exploration of hard-surface modeling techniques in Blender. The goal was to create a friendly, approachable robot character that could work in both stylized and semi-realistic contexts.

## Process

The modeling process began with basic shapes and gradually added detail through boolean operations and subdivision surface modifiers. The final model consists of approximately 50,000 polygons.

## Texturing

Textures were created in Substance Painter, using a combination of procedural materials and hand-painted details. The color palette was kept intentionally limited to maintain a cohesive look.

## Rigging

The character was rigged with a custom armature system that allows for both mechanical and slightly organic movement, giving the robot some personality in its animations.
