A procedural mushroom generation tool built in Houdini that creates a variety of stylized mushroom types with full control over shape, color, and clustering.

## Overview

This tool allows artists to quickly generate mushrooms ranging from realistic forest fungi to whimsical fantasy varieties. Parameters control everything from cap shape to stem texture to grouping behavior.

## Features

- Multiple mushroom type presets
- Customizable cap shapes and patterns
- Stem thickness and curvature controls
- Gill and spore detail generation
- Cluster and scatter placement tools
- Color variation and texture controls

## Technical Approach

The tool leverages Houdini's procedural workflow with VEX-driven shape generation. Artists can create single hero mushrooms or scatter thousands across a terrain with natural-looking variation.
