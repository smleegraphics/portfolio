A procedural tree generation system in Houdini specifically designed for creating apple trees with natural-looking branch structures and fruit distribution.

## Overview

This tool generates complete apple trees from trunk to fruit, with controls for tree age, branch density, leaf coverage, and apple placement. The system produces game-ready assets suitable for real-time rendering.

## Features

- L-system based branch generation
- Natural growth pattern simulation
- Procedural leaf placement and clustering
- Apple distribution with gravity-aware positioning
- LOD generation for game optimization

## Technical Details

The tool combines L-system algorithms with custom VEX code to simulate realistic tree growth. Branch thickness, angle variation, and tapering are all controllable through artist-friendly parameters.
