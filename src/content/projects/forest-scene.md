An atmospheric forest scene created for a game environment project. The goal was to achieve a magical, storybook aesthetic while maintaining reasonable polygon counts for real-time rendering.

## Design Philosophy

The scene uses a combination of hand-painted textures and stylized geometry to create a cohesive look. Trees, rocks, and foliage all follow the same visual language with soft gradients and bold color choices.

## Lighting

Lighting plays a crucial role in this scene. Volumetric fog and carefully placed point lights create depth and draw the eye through the composition. The main light source simulates late afternoon sun filtering through the canopy.

## Technical Details

- **Polygon Count:** ~25,000 triangles
- **Texture Resolution:** 1024x1024 atlases
- **Render Engine:** Cycles (for hero shots), Eevee (for real-time preview)
